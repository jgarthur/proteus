# legacy/python/

This folder contains an archived Python implementation and related tooling. It is kept for reference and is not the active implementation target.

## Key Folders

- `src/` - Python package source.
- `benchmarks/` - Benchmark, evaluation, sweep, and monitoring scripts.
- `tests/` - Test suite and golden test data.

## Key Files

- `pyproject.toml` - Python project metadata.
- `setup.py` - Build configuration for the Python package and optional extension paths.
- `uv.lock` - Locked Python dependency set.
